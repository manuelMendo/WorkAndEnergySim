# ⚛️ Virtual Physics Lab 2D

Interactive 2D virtual physics laboratory for simulating mechanics, collisions, forces, friction, and energy conservation in real time.

---

# 🚀 Features

- Real-time physics simulation
- Dynamic collision system
- Adjustable mass and velocity
- Friction and force calculations
- Energy conservation visualization
- Interactive mechanics engine
- 2D vector-based rendering
- Educational physics environment
- Responsive dashboard interface

---

# 🧠 Physics Concepts

This simulator demonstrates:

- Work
- Kinetic Energy
- Potential Energy
- Newton’s Laws
- Friction
- Momentum
- Elastic and Inelastic Collisions
- Force Interaction
- Acceleration
- Gravity Simulation

# ⚙️ Installation

Clone this repository:

```bash
git clone https://github.com/manuelMendo/virtual-physics-lab-2d.git
```

Open the project folder and run:

```bash
index.html
```
